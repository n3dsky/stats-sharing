# stats-sharing
# Анализ данных сервиса аренды самокатов GoFast

**Исследовательский анализ и проверка гипотез для роста бизнеса**

## Описание проекта

Проект посвящён анализу данных популярного сервиса аренды самокатов **GoFast**. 

Цель проекта — изучить поведение пользователей, проанализировать поездки и проверить статистические гипотезы, которые помогут бизнесу принимать обоснованные решения и увеличить выручку.

## Выполненные задачи

- Загрузка и предобработка данных
- Исследовательский анализ данных (EDA)
- Расчёт выручки сервиса
- Проверка статистических гипотез, важных для бизнеса

## Основные результаты и insights

В ходе анализа были выявлены следующие особенности:

- Самый представленный город — **Пятигорск**, меньше всего данных по **Москве**.
- Доля пользователей без подписки — **54.4%**, с подпиской — **45.6%**.
- Средний возраст пользователей — **25 лет** (от 12 до 43 лет).
- Средняя дистанция одной поездки — **3133 метра**.
- Пользователи без подписки чаще совершают короткие поездки и часто отменяют их сразу после оформления.
- Пользователи без подписки предпочитают более дальние поездки (3-й квартиль выше на 340 метров по сравнению с подписчиками).
- Пользователи с подпиской в среднем тратят больше времени на поездки.
- Средняя дистанция поездок у пользователей с подпиской составляет **3130 метров**.
- Пользователи с подпиской приносят сервису в среднем **больше дохода**, чем пользователи без подписки.

## Выводы

- Проведена качественная предобработка данных, подготовившая их к дальнейшему анализу.
- Выполнен глубокий исследовательский анализ поведения пользователей.
- Рассчитана выручка сервиса.
- Проверены ключевые гипотезы, интересующие заказчика.

Полученные insights позволяют лучше понимать поведение разных сегментов пользователей и могут быть использованы для разработки стратегий по увеличению удержания, конверсии в подписку и общей выручки сервиса.
